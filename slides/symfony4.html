Symfony 4

Incomming : symfony/flex

Plugin composer pour les projets symfony

But : simplifier et fluidifier la gestion des dépendances

Définition de recettes pour la configuration des bundles / librairies dans un fichier manifest.json

Utilisation de "configurateurs" pour gérer le comportement

L'exception Makefile

Aucune définition n'est nécessaire dans le manifest.json

La présence d'un fichier Makefile dans une recette entrainera la copie de son contenu dans le Makefile du projet

make your-own-recipes
